1. Format de fichier Excel attendu

Le fichier Excel doit contenir 2 onglets :
  • Onglet 1 : Balance N (période actuelle)
  • Onglet 2 : Balance N-1 (période précédente)

Colonnes attendues dans chaque onglet :

Numéro Intitulé Ant Débit Ant Crédit Débit Crédit Solde Débit Solde Crédit
101000 CAPITAL SOCIAL 0 100 000 000 0 14 116 101 0 114 116 101
111000 RESERVE LEGALE 0 20 000 000 20 000 000 0 0 0

Le solde net est calculé automatiquement : Solde Débit - Solde Crédit
